What is Better3.0?

Better3.0 -- is an extension for Chrome, that changes look of the scratch.mit.edu editor page. It brings some new features, such as dark mode in the editor, moving stage layout to the left side of the screen and more!

All the Options

Here are the list of options that you can enable or disable in Better3.0: • Move Stage to the Left Side of the Screen. • Enable Dark Mode in the Editor. • Increase Sprites List Area's Size by Shrinking an Info Bar. • (BETA) Show messages count in the editor. • Show remix tree button on the project pages • (BETA) Auto-hide blocks pallite, when no need
NOTE: If 'move stage to the left' option is enabled, moving scripts from one sprites to another through the sprites list area would be impossible. If you have this option enabled, please, use backpack! NOTE: Beta Options are the ones, that not tested enough/not completed. They can work not properly sometimes. Please, use it on your own risc!
